Java - HashMap

HashMap 介绍

HashMap 简介

HashMap 是一个散列表，它存储的内容是键值对 (key-value)映射。

HashMap 继承于 AbstractMap，实现了 Map、Cloneable、java.io.Serializable接口。

HashMap 的实现不是同步的，这意味着它不是线程安全的。它的 key、value 都可以为 null。此外，HashMap 中的映射不是有序的。
